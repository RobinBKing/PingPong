About Ping Pong

Created by:

Rob King

GitHub URL:

https://github.com/RobinBKing/PingPong.git

Description:

This application swaps out various numbers set by the user to a specified word. The project is a study in JavaScript, jQuery and Behavior Driven Development, (BDD).

Technology used:

Application: JavaScript, jQuery, HTML5, CSS3, bootstrap

Testing: Mocha, Chai

Want to play?

Here how it works:

  1. Numbers that can be divided by 3 get a 'Ping'
  2. Numbers that can be divided 5 get a 'Pong'
  3. Numbers that can be divided by 15 get a 'Ping Pong'
